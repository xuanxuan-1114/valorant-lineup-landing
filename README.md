# Valorant Lineup Assistant Landing Page

This is a public product introduction page for a Riot Developer Portal Production API Key application.

The site explains the product purpose, feature scope, user flow, compliance notes, privacy policy, and terms of service for Valorant Lineup Assistant. It is a static website prototype with no backend, no Riot API calls, and no API keys.

## Local Preview

You can preview the site in either of these ways:

- Double-click `index.html` to open it directly in a browser.
- Use VS Code Live Server and open the project folder.

## Deploy to GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder.
3. Open the repository `Settings`.
4. Go to `Pages`.
5. Select `main branch` and `/root`.
6. Save and wait for GitHub to publish the site.
7. Use the generated `github.io` URL as the public product page URL.

## Deploy to Vercel

1. Log in to Vercel.
2. Click `New Project`.
3. Import the GitHub repository.
4. Set `Framework Preset` to `Other`.
5. Click `Deploy`.
