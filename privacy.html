Privacy Policy

Last updated: June 8, 2026

Overview

Valorant Lineup Assistant is an educational lineup guide and training reference tool. The first version is designed to provide static tutorial content and does not require Riot account information.

Data Collection

  • No Riot API data is collected in the first version.
  • No personal Riot account data is collected in the first version.
  • No match history, ranked data, or live match data is collected in the first version.

Analytics

Basic website analytics may be added in the future only in a privacy-conscious way.

Future Account Features

If Riot Sign On or personal data features are added in the future, explicit user consent will be required.

Data Deletion

Users can contact the developer to request data deletion if future account features are added.
